    <section id="environmental-rules">
        <h3 style="color: #003366; border-left: 5px solid #003366; padding-left: 15px;">The Atmospheric Audit</h3>
        <ul style="margin-bottom: 25px;">
            <li><strong>The Dyspnea Scale Rule:</strong> Training intensity for pulmonary clients must be monitored using the 0-10 Borg Dyspnea Scale. Target missions are set between **3 (Moderate)** and **5 (Strong)**. Any reading above 6 requires an immediate reduction in workload.</li>
            <li><strong>The Heat Index Gate:</strong> If the local Heat Index exceeds **105&deg;F (or WBGT equivalent redlines)**, outdoor physical activity is redlined. Missions must be shifted to climate-controlled indoor environments.</li>
            <li><strong>The Medication Check (EIB):</strong> Clients with asthma must have their "rescue" inhaler accessible during all sessions. If an inhaler is forgotten, the mission is terminated.</li>
        </ul>
        
        <div style="background-color: #fef2f2; padding: 20px; border-left: 5px solid #dc2626;">
            <strong>Critical Logic:</strong> Oxygen patency overrides Intensity. In extreme heat, the cardiovascular system is already "Taxed" by thermoregulation, reducing the available reserve for aerobic performance.
        </div>
    </section>
