# EXW101 Module Audit & Alignment with Master Syllabus

## Course Learning Objectives (CLOs) from Master Syllabus:
- **CLO 2.1:** Recognize the physiological demands of varying levels of physical activity.
- **CLO 2.2:** Analyze fluid intake needed for various levels of physical activity. 
- **CLO 2.3:** List the nutritional requirements necessary for sports-related activities.
- **CLO 2.4:** Describe the relationship between diet and training for optimum performance.
- **CLO 2.5:** Discuss current theories related to weight gain and/or loss and peak performance. 
- **CLO 2.6:** Discuss the pros and cons of supplements.
- **CLO 2.7:** Describe the importance of timing when it comes to eating and fluid intake for optimal performance.

---

## Audit Summary:

This audit is based on the module titles and the learning objectives for Module 1. A more in-depth audit would require learning objectives for all modules.

### Strengths:

- **Strong Foundational Content:** The curriculum has strong coverage of macronutrients (Carbohydrates, Proteins, Lipids), micronutrients, hydration, and supplements, which aligns with CLOs 2.2, 2.3, 2.6 and 2.7.
- **Logical Structure:** The course structure with introductory modules, specialized topics, and a capstone project is logical and progresses from foundational knowledge to application.

### Gaps:

- **CLO 2.1 (Physiological Demands):** There is no module that explicitly addresses the physiological demands of varying levels of physical activity. This is a foundational concept that should be addressed early in the course.
- **CLO 2.5 (Weight Management):** There is no module that explicitly addresses weight gain and/or loss and peak performance. This is a significant gap in the curriculum.
- **CLO 2.4 (Diet and Training for Optimum Performance):** While several modules touch upon this, a dedicated module that synthesizes these concepts would be beneficial. The capstone may address this, but it is not explicit from the titles.

### Recommendations:

1.  **Add a New Module for CLO 2.1:** Create a new module early in the course (e.g., Module 2 or 3) that specifically addresses "Physiological Demands of Physical Activity". This module should cover topics like energy systems, muscle physiology, and the metabolic demands of different sports.
2.  **Add a New Module for CLO 2.5:** Create a new module that covers "Weight Management for Peak Performance". This module should discuss energy balance, body composition, and strategies for healthy weight gain and loss in athletes.
3.  **Create a Synthesis Module for CLO 2.4:**  Consider adding a module before the capstone that explicitly ties together diet and training for optimal performance, using case studies or practical examples.
4.  **Flesh out Module Learning Objectives:** The audit was hampered by the lack of explicit learning objectives for most modules. I recommend adding learning objectives to all modules to ensure and demonstrate alignment with the CLOs. This will also make the course easier for students to navigate and understand.
