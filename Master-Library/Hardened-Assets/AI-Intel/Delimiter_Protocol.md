# SOP: The Delimiter Protocol (### Triple-Hash ###)

## 1. Purpose
To ensure 100% data integrity when processing sensitive clinical metrics.

## 2. Standard Operating Procedure
All raw data must be explicitly wrapped in `###` characters.
### ✅ Correct
###
AGE: 64
BLOOD_PRESSURE: 168/104 mmHg
###

## 3. Scope
Mandatory for all graded clinical labs and peer audits.
