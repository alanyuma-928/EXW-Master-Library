## Caja de Herramientas de IA (AI Tool Kit)

### El Ciclo de la Misión (Mission Loop)
Para todas las tareas de EXW, utilizaremos el marco de **Patrón / Regla / Resolver**.

### 1. El Simulador de Clientes (Pattern)
* **Prompt:** "Actúa como un cliente de 45 años en Yuma que trabaja en el campo. Describe tus barreras para el ejercicio según el PAGA."

### 2. El Auditor de Reglas (Rule)
* **Prompt:** "Audita este plan de ejercicio usando el **PAGA 2018 (2nd Ed)** como la única fuente de verdad."

### 3. El Verificador de Matemáticas (Solve)
* **Prompt:** "Ayúdame a verificar los cálculos de vatios (watts) y gasto calórico para el cicloergómetro **Lode Corival**."

---
**Ejemplar para el Estudiante (Student Exemplar):**
> "Gemini, actúa como mi tutor clínico. Mi cliente tiene hipertensión. Según el PAGA, ¿cuál es la Regla (Rule) para el entrenamiento de fuerza?"
