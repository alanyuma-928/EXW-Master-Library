🏗️ The Architecture

High-Friction Learning & Clinical Infrastructure

The "Pilot" Philosophy

In the EXW residency, we reject the "Black Box" approach to AI. We do not use AI to simply generate answers; we use it to audit logic. This creates a "High-Friction" environment where the student remains the Pilot in Command.

1. The P-T-C-F Framework

Every student interaction with the "AI Engine" must utilize the P-T-C-F protocol to ensure clinical precision:

  • [P] Persona: Senior Clinical Exercise Specialist.
  • [T] Task: Conduct a clinical audit of this case study.
  • [C] Context: Reference the PAGA 2nd Ed (SSoT).
  • [F] Format: Markdown table with safety thresholds.

Sample P-T-C-F Prompt with Delimiters:

[PERSONA]: Senior Clinical Exercise Specialist.
[TASK]: Use the PAGA 2nd Ed (SSoT) to audit the client data provided below.
[FORMAT]: List specific modifications required for safety.

### CLIENT DATA BEGINS ###
Age: 68 | Condition: Hypertension & Osteoarthritis
Current Routine: 30min brisk walking, 4x weekly.
Goal: Add resistance training.
### CLIENT DATA ENDS ###

[INSTRUCTION]: Compare the routine inside the delimiters against the SSoT and identify any missing 'Rules' for resistance training.
🛠️ Architect's Note on Delimiters (###): We use triple-hashes as "Structural Bulkheads" to isolate raw client data from the Master Instructions. This prevents Context Drift and ensures the AI treats the data as a static reference point rather than a new set of commands.

2. The Mission Loop

We move students through a recursive cycle of Pattern / Rule / Solve to ensure workforce readiness:

  • Pattern: Identify the unique physiological markers of the client (The Data).
  • Rule: Match those markers against the Single Source of Truth / SSoT (The Regulation).
  • Solve: Generate a PAGA-aligned exercise prescription (The Output).

Institutional Agility

By using Infrastructure-as-Code (Markdown, GitHub, VS Code), this curriculum remains mobile-responsive and version-controlled. We prioritize tools that bypass administrative filters, ensuring students train on the actual clinical standards used in modern healthcare.
